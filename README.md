# Product_Sales_Analysis
Es un dataset que reúne datos de chaquetas Zara, incluyendo ubicación en tienda, promoción, temporada, material, origen, precio y ventas, para analizar cómo estas características influyen en el rendimiento comercial y la relación entre atributos del producto y resultados de ventas.

# Correlation between Price and Sales Volume for each Term:
Each terms has a similar negative correlation. That means there is not a particular term which is sensitive to the prices.  Price sensitivity appears consistent across product types, with correlations ranging from -0.32 to -0.37. This suggests that pricing impacts demand similarly across categories such as jackets, shoes, sweaters, jeans, and t-shirts.
