# Product_Sales_Analysis
Es un dataset que reúne datos de chaquetas Zara, incluyendo ubicación en tienda, promoción, temporada, material, origen, precio y ventas, para analizar cómo estas características influyen en el rendimiento comercial y la relación entre atributos del producto y resultados de ventas.

# Insights
Promotional products show 61.9% higher average sales volume, suggesting a strong positive association between promotions and demand.
End-cap placement may not justify additional merchandising cost based on current data.
Lower price → higher volume
